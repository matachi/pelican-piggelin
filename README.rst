========
Piggelin
========

Piggelin is a theme for `Pelican <http://getpelican.com>`_. I created it for
`my blog <http://piggel.in>`_.

License: `MIT <LICENSE>`_

Build
=====

Install dependencies:

.. code:: bash

    $ npm install -g grunt-cli
    $ npm install

Build:

.. code:: bash

    $ grunt

Continously build:

.. code:: bash

    $ grunt watch
